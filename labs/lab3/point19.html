Exemplu de pagină cu utilizarea funcțiunilor

Acest exemplu demonstrează utilizarea funcțiunilor JavaScript pentru a efectua o simplă adunare între două numere introduse de utilizator.

Codul demonstrativ:

Utilizăm HTML, CSS și JavaScript pentru a crea această funcționalitate.

Explicație:

În acest exemplu, am creat două câmpuri de input unde utilizatorul poate introduce două numere, iar prin apăsarea unui buton, se va calcula suma acestora. Funcția JavaScript calculeazaSuma() este apelată atunci când utilizatorul face clic pe buton.
