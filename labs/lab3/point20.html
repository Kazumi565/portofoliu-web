Exemplu de pagină cu obiectele Number, Math, Boolean, String

Acest exemplu demonstrează utilizarea obiectelor Number, Math, Boolean și String în JavaScript pentru a crea funcționalități interactive pe o pagină web.

Codul demonstrativ:

Utilizăm HTML, CSS și JavaScript pentru a crea această funcționalitate.

Explicație:

În acest exemplu, vom lucra cu patru obiecte fundamentale din JavaScript:

  • Obiectul Number: Generează un număr aleatoriu între 1 și 100.
  • Obiectul Math: Utilizează funcția Math.random() pentru a genera un număr aleatoriu.
  • Obiectul Boolean: Verifică o expresie logică bazată pe un număr aleatoriu și returnează true sau false.
  • Obiectul String: Modifică un șir de caractere pentru a-l transforma în majuscule.
