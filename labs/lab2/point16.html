Exemplu de integrare a expresiilor de scriptare

Acest exemplu arată cum poți utiliza expresii ale limbajului de scriptare pentru a genera conținut dinamic într-o pagină web. Vom folosi PHP pentru a insera un mesaj personalizat în cadrul HTML-ului.

Cod demonstrativ:


<?php
  // Setăm o variabilă cu un mesaj
  $salut = "lumea";
  
  // Afișăm un titlu dinamic
  echo "<h1>Salut, $salut!</h1>";
?>

Explicație:

În exemplul de mai sus, codul PHP este inclus între tag-urile <?php și ?>. Variabila $salut este setată cu textul "lumea", iar funcția echo este folosită pentru a insera în pagină un tag HTML <h1> care afișează mesajul "Salut, lume!".

Astfel, dacă fișierul este procesat de un server web cu suport PHP, rezultatul va fi o pagină web dinamică în care conținutul poate fi modificat de logica din spatele codului. Această metodă este esențială pentru crearea de site-uri dinamice care se adaptează în funcție de datele introduse, utilizator sau alte condiții de mediu.

Reține că pentru ca acest cod să funcționeze corect, fișierul trebuie să fie salvat cu extensia .php și să fie rulat pe un server care interpretează PHP.
