Exemplu de Machetare Responsivă

În acest punct, vom explora cum putem implementa un design responsive folosind media queries în CSS. Această tehnică asigură că pagina se adaptează la diferite dimensiuni de ecran, îmbunătățind experiența utilizatorului pe dispozitive mobile.

Cod demonstrativ:

/* Stilurile de bază pentru ecrane mari */
body {
  font-size: 16px;
  background-color: #333;
  color: white;
}

/* Media Query pentru ecrane mai mici de 600px */
@media only screen and (max-width: 600px) {
  body {
    background-color: #222;
    font-size: 14px;
  }

  header, nav, main, footer {
    padding: 10px;
  }

  nav ul {
    flex-direction: column;
    align-items: center;
  }
}

Explicație:

În exemplul de mai sus:

  • Stiluri de bază: Se setează un font de 16px și un fundal de culoare gri închis pentru ecrane mai mari.
  • Media Query: Pentru ecranele cu lățimea sub 600px, se schimbă culoarea de fundal la o nuanță mai închisă, se reduce dimensiunea fontului la 14px și se ajustează padding-ul pentru a îmbunătăți aspectul și lizibilitatea.
