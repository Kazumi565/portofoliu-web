Ce sunt instrucțiunile decizionale?

Instrucțiunile decizionale permit unui program să ia decizii pe baza anumitor condiții. În JavaScript, cele mai comune instrucțiuni sunt:

  • if – execută un bloc de cod dacă o condiție este adevărată.
  • if-else – execută un bloc de cod dacă condiția este adevărată și un altul dacă este falsă.
  • else if – permite mai multe condiții alternative.
  • switch – este utilizat pentru a compara o valoare cu mai multe cazuri.

Exemplu interactiv

Mai jos este un exemplu care afișează un mesaj diferit în funcție de ora curentă:

  • Dacă ora este mai mică de 12:00, se afișează mesajul „Bună dimineața!”
  • Dacă ora este între 12:00 și 18:00, se afișează mesajul „Bună ziua!”
  • Dacă ora este mai mare de 18:00, se afișează mesajul „Bună seara!”

Scriptul folosește obiectul Date pentru a obține ora curentă și apoi decide ce mesaj să afișeze pe pagină utilizând if-else.

Explicație cod

Scriptul JavaScript verifică ora curentă a sistemului și afișează un mesaj corespunzător: 


let ora = new Date().getHours();

if (ora < 12) {
    console.log("Bună dimineața!");
} else if (ora < 18) {
    console.log("Bună ziua!");
} else {
    console.log("Bună seara!");
}

- Variabila ora stochează ora curentă. - Prima condiție (if) verifică dacă ora este mai mică de 12 și afișează "Bună dimineața!". - A doua condiție (else if) verifică dacă ora este între 12 și 18, afișând "Bună ziua!". - Dacă niciuna dintre condițiile anterioare nu este adevărată, se afișează "Bună seara!".
