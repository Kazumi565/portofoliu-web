Utilizarea Metodelor de Stocare a Datelor

  • Acasă

Introducere

În această pagină, vom explora diferitele metode de stocare a datelor în aplicațiile web și cum să utilizăm fiecare dintre acestea.

Metoda 1: localStorage

localStorage permite stocarea de date în browser, iar datele rămân disponibile chiar și după închiderea browserului.

        // Stocarea datelor
        localStorage.setItem('username', 'Mihai');

        // Recuperarea datelor
        const username = localStorage.getItem('username');
        console.log(username);  // Mihai

Metoda 2: sessionStorage

sessionStorage stochează date doar pentru durata sesiunii curente a utilizatorului (se șterge când se închide fereastra).

        // Stocarea datelor
        sessionStorage.setItem('sessionID', '12345');

        // Recuperarea datelor
        const sessionID = sessionStorage.getItem('sessionID');
        console.log(sessionID);  // 12345

Metoda 3: Cookies

Cookie-urile sunt folosite pentru stocarea unor informații care pot fi trimise către server cu fiecare cerere.

        // Setarea unui cookie
        document.cookie = "user=John; expires=" + new Date(2025, 1, 20).toUTCString();

Metoda 4: IndexedDB

IndexedDB este o metodă avansată de stocare a datelor complexe, potrivită pentru aplicațiile care necesită stocarea unor cantități mari de date.

        // Exemplu de utilizare a IndexedDB
        let request = indexedDB.open("myDatabase", 1);
        request.onsuccess = function(event) {
            let db = event.target.result;
            let transaction = db.transaction("myStore", "readwrite");
            let store = transaction.objectStore("myStore");
            store.add({ name: "Mihai", age: 18 });
        };

Metoda 5: Stocarea pe Server

În aplicațiile reale, datele sunt adesea stocate pe un server folosind o bază de date externă, accesată prin API-uri.
