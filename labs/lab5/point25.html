Gestionarea Obiectelor Modelului BOM

În această secțiune, vom explora cum să gestionăm obiectele din modelul BOM (Bill of Materials) folosind o aplicație web simplă. Vom crea un formular pentru a adăuga obiecte în lista BOM și vom putea vizualiza și șterge obiectele deja adăugate. Vom folosi JavaScript pentru a manipula aceste date dinamic pe pagină.

Ce este BOM (Bill of Materials)?

Modelul BOM (Bill of Materials) este o listă detaliată a materiilor prime, componentelor, subansamblelor, pieselor și cantităților necesare pentru fabricarea unui produs sau pentru a construi o structură complexă. BOM este utilizat pe scară largă în industrie, în special în fabricarea de produse, construcția de echipamente electronice, și chiar în gestionarea proiectelor de construcții.

Unde este utilizat BOM?

BOM-ul este utilizat în multe industrii, inclusiv:

  • Fabricarea de produse: Un BOM detaliază toate piesele necesare pentru asamblarea unui produs final, cum ar fi o mașină sau un computer.
  • Industria electronică: În producția de dispozitive electronice, BOM-ul este esențial pentru gestionarea componentelor electrice și a circuitelor.
  • Construcția: În construcția de clădiri, BOM-ul ajută la urmărirea materialelor și pieselor necesare pentru construcție.

În acest laborator, vom construi o aplicație simplă care simulează gestionarea unui BOM pentru un proiect de mică amploare, adăugând obiecte și cantități într-o listă și manipulând aceste date dinamic pe pagină.

Cum funcționează aplicația?

Aplicația noastră va permite utilizatorilor să adauge obiecte într-o listă BOM și să le vizualizeze. De asemenea, vom implementa opțiunea de a șterge obiectele deja adăugate. Vom folosi HTML pentru structura formularului, CSS pentru stilizarea paginii, și JavaScript pentru a manipula datele și a actualiza lista dinamic.

Pașii pentru crearea aplicației

  1. Crearea formularului: Vom crea un formular simplu care va permite utilizatorilor să adauge obiecte în lista BOM, completând câmpurile pentru numele obiectului și cantitatea acestuia.
  2. Gestionarea obiectelor cu JavaScript: Vom folosi JavaScript pentru a colecta datele introduse de utilizatori, a le adăuga într-o listă și a le vizualiza pe pagină. De asemenea, vom implementa o funcție de ștergere pentru a elimina obiectele care nu mai sunt necesare.
  3. Actualizarea listei dinamic: Când utilizatorul adaugă un obiect, acesta va apărea imediat în lista vizibilă pe pagină, iar utilizatorul va putea șterge obiectele din listă cu un click.

Explicație detaliată a funcționalităților

În această aplicație, am folosit un formular HTML simplu care colectează date despre obiectele BOM. Când utilizatorul trimite formularul, JavaScript preia valorile introduse pentru numele și cantitatea obiectului, le adaugă într-un array (listă) și actualizează pagina pentru a afișa lista actualizată de obiecte.

De asemenea, fiecare obiect din listă are un buton "Șterge" care, atunci când este apăsat, elimină obiectul respectiv din listă. Aceste funcționalități sunt gestionate printr-un cod JavaScript care manipulează DOM-ul paginii pentru a adăuga și elimina elemente vizibile.

Codul JavaScript utilizat

În acest exemplu, am folosit următoarele concepte în JavaScript:

  • Evenimente: Folosim un eveniment pentru a detecta atunci când utilizatorul trimite formularul și adaugă un obiect în listă.
  • Manipularea DOM-ului: După adăugarea obiectului, folosim JavaScript pentru a crea un nou element în HTML și a-l adăuga în lista vizibilă pe pagină.
  • Array: Obiectele sunt stocate într-un array și sunt gestionate dinamic pe măsură ce utilizatorul interacționează cu aplicația.
