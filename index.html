Laborator 1

Acest laborator explorează elementele de bază ale paginilor web.

Puncte din laborator
  • 1. Inserarea blocurilor de text
  • 2. Inserarea listelor
  • 3. Inserarea referințelor
  • 4. Inserarea imaginilor
  • 5. Inserarea elementelor multimedia
  • 6. Inserarea cadrelor interne
  • 7. Inserarea tabelelor
  • 8. Inserarea formularelor
  • 9. Formatarea blocurilor de text
  • 10. Poziționarea elementelor
  • 11. Utilizare pseudo-clase și pseudo-elemente
  • 12. Formatarea elementelor multimedia
  • 13. Crearea elementelor de navigare

Laborator 2

Acest laborator abordează teme avansate de dezvoltare web.

Puncte din laborator
  • 14. Machetarea responsivă a paginii web
  • 15. Publicarea paginilor web
  • 16. Pagini web cu expresii ale limbajului de scriptare
  • 17. Pagini web cu instrucțiuni decizionale

Laborator 3

Acest laborator se concentrează pe utilizarea JavaScript în paginile web.

Puncte din laborator
  • 18. Pagini web cu instrucțiuni repetitive
  • 19. Pagini web cu utilizarea funcțiunilor
  • 20. Pagini web cu obiectele Number, Math, Boolean, String

Laborator 4

Acest laborator abordează prelucrarea tablourilor, utilizarea evenimentelor și comportamentele client-side ale elementelor din documente web.

Puncte din laborator
  • 21. Pagini web de prelucrarea a tablourilor
  • 22. Pagini web cu utilizarea evenimentelor mouse-ului, tastaturii, ferestrei browser-ului
  • 23. Scripturi client-side pentru definirea comportamentului elementelor dintr-un document web

Laborator 5

Acest laborator se concentrează pe verificarea datelor din formular, gestionarea obiectelor BOM și utilizarea metodelor de stocare a datelor.

Puncte din laborator
  • 24. Pagini web cu verificarea datelor din formular
  • 25. Pagini web de gestionare a obiectelor modelului BOM
  • 26. Pagini web cu utilizarea metodelor de stocare a datelor
